# Epistemic Kernel Laboratory v0.1

**The smallest possible implementation of an Evidence-First scientific reasoning system.**

This repository contains the canonical minimal viable proof that:

> **Evidence is immutable. Reasoning engines are disposable. Knowledge is derived and replayable.**

It is the foundational kernel for the larger Codessa / CEOS vision — an operating system for building and improving knowledge from evidence. The music-specific work (SAL, MAL, PROMETHEUS, MUSE) is one domain adapter built on top of this kernel.
